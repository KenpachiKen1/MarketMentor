#
**App Name: Market Mentor**

##
Market Mentor is a full stack learning platform to help beginners understand stock market concept through modules, quizzes, stock lookup and portfolio management. The applications has educationcal content (with videos
articles, and quizzes) and live news/financial data. Users progress through these modules to unlcok a the gold theme background once they complete all 3 modules.
###
Target Browsers:
Market Mentor is a web based application, **Google Chrome is the only working browser as of right now.** Mobile support has not been added yet, while the frontend will load on mobile, the layout will be a complete mess as of right now.
